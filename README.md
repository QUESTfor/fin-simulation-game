一新投資理財遊戲

零、連結
1. 遊戲本體連結: https://fin-simulation-game-oqbsku4hbaz4w3ihkrjzau.streamlit.app/
2. 組長們確認是否已繳交用: https://questfor.github.io/student-check/

一、遊戲概念
1. 學生原先有資產30000，將面臨生活、飲食、投資三大開銷，其中生活跟飲食隨著遊戲設定事件而變更，例如通膨或關稅，學生不可更改。
2. 學生每月獲得主動收入10000，並遭遇預先設定好的每月事件，去判斷投資操作，投資操作共有儲蓄、基金、債券跟股票(又分房地產跟高利率的生醫股)、借款。
3. 設定好的每月事件為一連串敘事，需要設計為想傳達的投資理念，舉例如下:  (見p5)
-設定共同基金高昂管理與手續費，並使一年的報酬不如ETF大盤(主動型基金打不贏被動型: https://longbets.org/362/)
-升息促使學生選擇儲學生選擇儲蓄，利空股市
-設定高利率的生醫股伴隨高風險，五月暴漲，而九月被發現造假數據，持股變一文不值
-使升息伴隨著高通膨，提高房地產，傳達並不是只要升息就是利空，也有可能使更多人買房抵制通膨，經濟並不是簡單因果而是綜合各變項的學門。
-戰爭、貿易戰、產生通膨影響
-賭博bad(期望值負的)
